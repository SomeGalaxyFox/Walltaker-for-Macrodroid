# Walltaker-for-Macrodroid
Walltaker Client Macro for Macrodroid

## Installation
1. Download Walltaker_Client.macro
2. Import into Macrodroid
3. Set Variables
  - Walltaker link (just the ID)
  - Screen on interval (in seconds)
  - Screen off interval (in seconds)
4. Activate Macro or change trigger if desired
